-
-　サウンドファイルのループ範囲情報ファイルを作成するソフト
-
-　Version 1.0
-
-----------------------------------------------------------------

・概要

　　　サウンドファイルのループ再生する範囲を編集、保存するためのソフトです。

　　　ゲームのＢＧＭは前奏部分とループ再生される部分がひとつのサウンドファイルに
　　なっている場合が多いですが、Oggファイルなどを除いてループ再生する範囲を
　　ファイルの段階で指定する方法が無く、プログラム内部でループの範囲を指定する
　　必要があります。

　　　ですが、ＤＸライブラリではサウンドファイル名の末尾が _loop.txt となっている
　　テキストファイルに記載されたループ再生の範囲情報を自動的に読み込み適用する
　　機能があります。( Ver3.16d 以降 )

　　　そして、このソフトはその「サウンドファイル名の末尾が _loop.txt のファイル」を
　　作成するためのソフトです。(「サウンドファイル名の末尾が _loop.txt のファイル」
　　は一般的なテキストファイルなので、ループの範囲が分かっている場合は一般的な
　　テキストエディタで直接 _loop.txt のファイルを作成することもできます )


・ファイル説明

　　SoundLoopAreaEditor.exe
　　　　　サウンドファイルのループ範囲情報ファイルを作成するソフトです


・使い方

　　　SoundLoopAreaEditor.exe を起動するとサウンドファイルをドラッグ＆ドロップするように
　　促されますので、ＤＸライブラリが対応しているサウンドファイルをドラッグ＆ドロップしてください。

　　　するとループ再生範囲の編集モードになります。

　　　このソフトは全てキーボードで操作します。
　　　スペースキーでサウンドの再生、方向キーの←、→でカーソルを移動し、Sキーでループ範囲の
　　開始点の設定、Eキーでループ範囲の終了点の設定、Wキーでループ範囲情報の保存、といった具合です。
　　全ての操作説明は常にウインドウ内に書かれていますので、操作方法が不明な場合は説明を見てください。

　　　ソフトでやるべきことは単純で、ループ再生する範囲を指定して、Wキーでファイルを保存する
　　ことです。

　　１．最初にスペースキーで普通に再生して、どの辺りがループ再生の範囲かを把握する

　　２．ループ再生の範囲を Sキーと Eキーで大雑把に設定する

　　３．方向キーの↑キーと↓キーの波形拡大機能と、Dキーのループ直前からの再生を利用しながら
　　　　ループ再生の範囲を細かく調整する

　　４．ループ再生の範囲が決定したら Wキーでループ範囲の情報である _loop.txt のファイルを保存する


・ループ再生範囲の情報ファイルをプログラムで使用する方法

　　LoadSoundMem でサウンドファイルを読み込むと自動的に末尾が _loop.txt となっているファイルから
　　ループ範囲の情報を読み込むようになっていますので、プログラムでは特に何の処理も加える必要は
　　ありません。( ただしＤＸライブラリのバージョンが 3.16d 以降である必要があります )
