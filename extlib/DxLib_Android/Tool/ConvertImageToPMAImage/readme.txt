・概要

　　　通常の画像から乗算済みアルファの画像を作成するソフト。

　　　『乗算済みアルファ』についてはＤＸライブラリパッケージファイル内の
　　help\lecture\PremulAlpha\PremulAlpha.html を参照してください。

　　↑をご覧になられた前提で説明をします。

　　『乗算済みアルファ』を使用する場合は画像を読み込む際に予め
　　SetUsePremulAlphaConvertLoad( TRUE ); を呼んでおく必要がありますが、
　　これを実行することで行われる処理は

　　「『通常画像』を読み込んだ後に『乗算済みアルファの画像』に変換する」

　　です。
　　そして、この『乗算済みアルファの画像』とは、単純に色の値を変更しただけの
　　画像なので、pngファイルや tgaファイルの状態でも『乗算済みアルファの画像』に
　　することができます。

　　　そして、このソフトはその
　　「pngファイルや tgaファイルのまま『乗算済みアルファの画像』に変換する」
　　ためのソフトです。

　　pngファイルや tgaファイルのまま『乗算済みアルファの画像』にするメリットは
　　二つあります。

　　1.SetUsePremulAlphaConvertLoad( TRUE ); を実行する必要がなくなる

　　　　SetUsePremulAlphaConvertLoad( TRUE ); は『通常の画像』を LoadGraph で
　　　読み込んだ際などに『乗算済みアルファの画像』に変換する処理を行うように
　　　するために実行するものなので、pngファイルや tgaファイルの時点で
　　　『乗算済みアルファの画像』となっている場合は実行する必要がなくなります。
　　　　これによって『通常の画像』から『乗算済みアルファの画像』に変換する
　　　処理の分だけ画像ファイルの読み込み時間が短縮できます。

　　2.DXT圧縮の画像のまま『乗算済みアルファ』が使用できる

　　　　DXT圧縮された画像ファイル( 拡張子dds )を SetUsePremulAlphaConvertLoad( TRUE );
　　　を実行した状態で読み込むと、DXT圧縮されたままの状態では『乗算済みアルファ
　　　の画像』に変換できないため、まず『無圧縮32bitカラー画像』に変換された後、
　　　『乗算済みアルファの画像』に変換されます。
　　　　この場合、DXT圧縮の最大の利点である『画像のデータ容量が少ない』が無くなって
　　　しまいます。
　　　　ですが、このソフトを使用して予め『乗算済みアルファの画像』にした pngファイルを
　　　DXT圧縮の画像ファイルに変換すれば SetUsePremulAlphaConvertLoad( TRUE ); を
　　　使用する必要が無いので、DXT圧縮の利点を生かしたまま『乗算済みアルファの画像』
　　　として使用することができます。


・ソフトの使い方

　　＜GUI版としての使い方＞

　　　　ConvertImageToPMAImage.exe を起動すると

　　『乗算済みアルファ画像に変換したい画像をドラッグ＆ドロップしてください』

　　　　と書かれたウィンドウが表示されますので、書いてある通りに『乗算済みアルファの画像』に
　　　変換したい『通常の画像』の pngファイル、tgaファイルをウィンドウに
　　　ドラッグ＆ドロップしてください。

　　　　ファイル名の末尾に『_PMA』が追加された『乗算済みアルファの画像』に変換された
　　　pngファイルが作成されます。( PMA は『Pre Multiply Alpha』の頭3文字です )


　　＜CUI版としての使い方＞

　　　　ConvertImageToPMAImage.exe はコンソールアプリとしても実行できます。

　　　ConvertImageToPMAImage.exe  変換するファイルのパス  [変換後のファイルのパス]

　　　　[変換後のファイルのパス]が省略されると変換するファイルのパスのファイル名の
　　　末尾に『_PMA』が追加されたファイル名で保存されます。


・作成した『乗算済みアルファの画像』の使い方

　　　　通常の画像と同じように LoadGraph で読み込みます。
　　　　ただ、前述の通り SetUsePremulAlphaConvertLoad( TRUE ); を実行していなくても
　　　『乗算済みアルファの画像』となっていますので、描画時は描画ブレンドモードを
　　　DX_BLENDMODE_PMA_ALPHA などの『乗算済みアルファ』用のものを使用する必要があります。
